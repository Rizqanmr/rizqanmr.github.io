Sistem Informasi Inventaris Kelas

Sistem Informasi Inventaris Kelas ini merupakan project tugas akhir kelompok saya pada mata kuliah Proyek Kekhususan TIK. Sistem ini terdiri dari dua platform yaitu Android dan Web. Pada platform android, diperuntukan untuk 3 jenis user seperti Dosen/Mahasiswa, Petugas, Kajur. Sedangkan untuk backend-nya kami menggunakan platform website dengan framework laravel, yang dimana hanya admin yang dapat menggunakan web ini untuk proses CRUD data. Agar dapat meng-handle semua hal yang berkaitan dengan koneksi data dari android ke website, kami menggunakan library di pemrograman android yaitu Retrofit. Dan dalam pengembangan sistem ini kami menggunakan metode Agile Development dengan framework Scrum.

  • Fitur-fitur Web:
  • - Login Admin
  • - Create, Read, Update dan Delete
  • Fitur-fitur Android:
  • - Login sesuai dengan jenis user (Dosen/Mahasiswa, Petugas, Kajur)
  • - Dosen/Mahasiswa dapat melihat kelas dan barang jurusan
  • - Dosen/Mahasiswa dapat Membuat laporan kerusakan barang
  • - Dosen/Mahasiswa dapat meminjam barang (dalam pengembangan)
  • - Kajur dapat melihat laporan kerusakan barang dan menugaskan kepada petugas untuk segera diperbaiki (dalam pengembangan)
  • - Petugas dapat mengupdate laporan kerusakan barang (dalam pengembangan)

  • Date: 27 April 2020
  • Category: Android (Java) dan Web (Laravel)
